This script downloads YouTube videos and converts to mp3.
It will remove the mp4 YouTube downloaded file


Install Python from:
		https://www.python.org/downloads/
#### To check if you have python
#### Open command prompt and type `python --version`


Install pip packages with command:
		`pip install -r requirements.txt`
#### Note: if you get an error
#### open command prompt or powershell in administrator mode
#### Then re-run command 

run python script:
		`python ytmp3.py [YOUTUBE VIDEO URL HERE]`
